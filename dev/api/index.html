WaveOperators.IntegralEquation — Type
IntegralEquation

This object represents the physics equation

\[z + G\mathbf{diag}(\theta)z = Gb,\]

in compact form. Specifically, G_design maps from the design region to the design region, and G_target maps from the design region to the target. By only dealing with the reduced form of the system, we avoid unnecessary computation. The reduced form of the physics is

\[\begin{aligned} z^\mathrm{design} + G^\mathrm{design}z^\mathrm{design} &= b^\mathrm{design}\\ z^\mathrm{target} + G^\mathrm{target}z^\mathrm{design} &= b^\mathrm{target} \end{aligned}\]
