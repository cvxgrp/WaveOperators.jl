WaveOperators

Overview

This package contains utility functions to setup physical design problems[1], using an integral equation approximation to the Helmhotlz equation. While it was developed for computational bounds in photonics[2], we believe that these operators may be useful for anyone working on problems in physical design. To get started, check out the examples!

For more details on the methodology and performance tricks, please see appendices A & B of Bounds on Efficiency Metrics in Photonics.
